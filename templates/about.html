Statuspy

What is Statuspy?

In short: another Twitter-clone. It is Python + Tornado + Redis powered and written by Thomas Pelletier. Statuspy is not meant to replace Twitter whatsoever. I built it as a backup for my Twitter feed, and add it some of the features I miss in Twitter. The source code is obviously available on GitHub and it is licensed under the MIT license. Feel free to fork it.
